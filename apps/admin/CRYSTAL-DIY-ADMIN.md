# 一念一串后台管理接入说明

本目录基于 Vben Admin 5 初始化，作为一念一串管理后台基座。

## 后端接口

- 本地 API：`http://localhost:3000/api`
- Swagger：`http://localhost:3000/api/docs`
- 管理员登录：`POST /api/admin/auth/login`
- 当前管理员：`GET /api/admin/auth/me`

## 一期菜单规划

- 工作台：销售概览、待发货、库存预警、待审核念卡
- 商品中心：素材库、SKU、库存流水、价格规则
- 主题推荐：五行、星座、MBTI、生肖/本命年规则
- DIY 管理：设计草稿、报价记录、可制作性校验
- 订单中心：订单、支付流水、退款、售后、物流、评价
- 手串册：成品记录、公开审核
- 念卡内容：念卡记录、模板管理、9:16 海报模板
- 用户中心：小程序用户、地址、收藏
- 系统管理：管理员、角色、权限、审计日志
